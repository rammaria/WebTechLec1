The components of the website are saved in the "website" folder that has subfolders. The "html" subfolder contains the different HTML documents. The "style" subfolder contains the CSS files. The HTML and CSS files are already linked together based on the current directory structure.

To access the website, follow the steps below:
1. Open the "website" folder.
2. Open the "html" folder.
3. Using a web browser, open "index.html". That should display the homepage of the website.
